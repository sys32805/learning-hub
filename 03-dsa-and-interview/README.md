## Study flow

1. **oops** — four pillars, abstract vs interface, SOLID (interview theory)
2. **concepts** — pick one topic (e.g. arrays / trees)
3. **algorithms** — read + rewrite 1–2 Java solutions in that package
4. **04-leetcode** — solve 1–2 matching problems

Stay in Java end-to-end; this track is for backend / interview prep.
